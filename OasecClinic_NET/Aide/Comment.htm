Configuration logicielle minimale requise :

  • Windows 2000
  • Internet Explorer (IE 5.5)
  • Internet Information Server (IIS 5)
  • .Net Framework 1.0
  • SQL Server ou MSDE 2000

Services Internet (IIS)

Cliquez sur Démarrer, pointez sur Paramètres, cliquez sur Panneau de configuration, et démarrez l'application Ajout/Suppression de programmes.

Sélectionnez Ajouter/Supprimer des composants Windows, Cochez sur Services Internet (IIS).

Installation manuelle du Site

  1. Décompressez le fichier Projet.exe
  2. Dans le fichier "Clinique_Base.sql"
    • Remplacez "<Data>" par le chemin de base de données de votre serveur MSSQL
    • Remplacez "<Domaine>" par le nom de votre ordinateur
  3. Dans la ligne de command
    • Lancez la commande "osql -E -i Clinique_Base.sql -o Clinique_Base.txt"
      (consultez le fichier Clinique_Base.txt pour assurer l'importation de la base)
  4. Partage du répertoire web
    • Dans le Gestionnaire des services Internet
    • Choisir Site Web par défaut
    • Cliquez sur Nouveau répertoire virtuel (Fig 3)
    • Donnez "Clinique" comme alias (Fig 4)
    • Spécifiez le chemin du site (Fig 5)
